Important: Please read this first before creating data

Before creating any data, please carefully review all guidelines below to ensure your annotations meet the quality standards and cultural authenticity requirements.

Annotation Guidelines

Data Sources

  • Multilingual Data: Based on the annotated data from other language, could you remind any cultural-related scenarios in your culture? If yes, please create it.

Creating Authentic Cultural Data

Draw from authentic sources: Personal experiences, local forums (Reddit, community Q&A), cultural events you've attended, and community observations.

Ensure cultural authenticity: Use specific local details (places, customs, terminology) that only locals would know, avoiding stereotypes and generic scenarios.

Create clear, unambiguous questions: Each scenario should have one definitive, culturally-informed answer with sufficient context to feel realistic.

Making It Challenging for AI

  • Use specific entities: Replace generic terms with local places, brands, or cultural-specific terminology
  • Include implicit knowledge: Reference cultural norms understood locally but not explicitly stated
  • Add temporal context: Include seasonal, holiday, or time-sensitive cultural references
  • Incorporate social nuances: Use subtle social dynamics, hierarchies, or etiquette rules
  • Require multi-step reasoning: Create scenarios needing conditional cultural logic
