Multilingual Format

Make It Challenging!

Create questions that are as challenging as possible for AI models:

  • Long-Tail Swap: Replace common entities with rarer, more specific ones (e.g., "Hong Kong" → "MacLehose Trail")
  • More Context: Add situational details requiring conditional, multi-step reasoning (e.g., visa requirements with prior conditions)
  • Less Context: Remove unnecessary hints that could help LLMs guess the answer
  • Compositional Example: Combine two independent knowledge points into a single query (e.g., entry requirements for both Hong Kong and Bangkok)
